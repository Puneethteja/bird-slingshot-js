A physics-based, mobile-responsive projectile game built using **p5.js** and **Matter.js**. 

Inspired by classic physics games, this project allows players to launch birds from a slingshot to clear levels by destroying targets.

## 🚀 Features
* **Physics Engine:** Built on the robust Matter.js engine for realistic collisions and gravity.
* **Mobile-Optimized:** Designed for touchscreens with responsive scaling and custom touch-event handling.
* **Dynamic UI:** Includes an interactive start screen, instructions, and win/loss states.
* **Flight Mechanic:** Unique "tap-to-fly" mid-air boost mechanic.
* **Time-Aware:** Features a dynamic background that changes based on the time of day.

## 🎮 How to Play
1. **Start:** Click/Tap the "TAP TO START" button on the main screen.
2. **Aim:** Drag the bird back on the slingshot and release to launch.
3. **Boost:** Tap anywhere on the screen while the bird is in mid-air to gain a velocity boost.
4. **Reset:** Press the `SPACE` key (or restart via the game-over screen) to reset the bird's position.

## 🛠️ Built With
* [p5.js](https://p5js.org/) - Creative coding library.
* [Matter.js](https://brm.io/matter-js/) - 2D physics engine.
* [WorldTimeAPI](http://worldtimeapi.org/) - For dynamic background scheduling.

## 📋 Installation
1. Clone this repository:
   `git clone https://github.com/Puneethteja/bird-slingshot-js.git`
